
About Our Stack:

Our Stack Utilizes:
	Node.JS (Server), 
	Locomotive (NodeJS MVC built on Express), 
	Sequelize (ORM for RDBMS),
	Faker (Test Data Generator),
	Mongoose (ODM for MongoDB), 
	Passport (Auth Middleware for Express (and thus Locomotive),
	AngularJS (Client side MVC),
	TwitterBootstrap (Web UI Framework)

Locomotve:
 Must be installed globally with root privileges, not as part of package json
